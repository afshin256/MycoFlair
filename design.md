# MycoFlair.ir Design System

## Design Philosophy
- Modern, clean, and professional aesthetic
- Light theme with nature-inspired colors
- Smooth animations and micro-interactions
- Floating cards with depth and shadow effects
- Responsive design for all screen sizes
- Bilingual typography considerations

## Color Palette
- **Primary Green**: #2D5A27 (Deep forest green for headers and accents)
- **Secondary Green**: #4A7C59 (Medium green for buttons and highlights)
- **Light Green**: #8FBC8F (Soft green for backgrounds and cards)
- **Cream**: #F5F5DC (Background and card backgrounds)
- **Soft Yellow**: #F0E68C (Accent color for highlights)
- **White**: #FFFFFF (Primary background)
- **Dark Gray**: #2C3E50 (Primary text color)
- **Medium Gray**: #5D6D7E (Secondary text color)

## Typography
- **Display Font**: 'Playfair Display' (serif) for headings
- **Body Font**: 'Inter' (sans-serif) for body text
- **Persian Font**: 'Vazirmatn' for Persian text
- **Font Sizes**: 
  - H1: 3rem (48px)
  - H2: 2.25rem (36px)
  - H3: 1.875rem (30px)
  - Body: 1rem (16px)
  - Small: 0.875rem (14px)

## Visual Effects
- Floating cards with hover lift effects
- Smooth scroll animations
- Gradient backgrounds
- Subtle parallax effects
- Animated counters
- Typewriter effects for headings
- Image carousels with Ken Burns effect

## Layout System
- Container max-width: 1200px
- Grid system with 12 columns
- Responsive breakpoints:
  - Mobile: 320px - 768px
  - Tablet: 768px - 1024px
  - Desktop: 1024px+
- Section padding: 4rem vertical, 2rem horizontal
- Card padding: 2rem
- Border radius: 12px for cards, 8px for buttons
